# Dairy Farm App

A complete Flutter desktop application for managing dairy farm operations.

## Features

- Animal Management
- Milk Production Tracking
- Client Management
- Sales & Billing
- Credit Handling
- Expense Tracking
- Ledger System
- Vaccination Records
- Excel Export
- Dashboard Overview

## Technology

- Flutter (Windows Desktop)
- SQLite Database
- GetX State Management
